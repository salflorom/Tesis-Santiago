\chapter{Resultados}    
    \noindent Para analizar los resultados, se definió la diferencia de entropías, donde se toma como referencia aquellas asociadas a la base 6-31G,
    \begin{align}
        \Delta (S_{\delta}^{\tau})_i^{6-31G} \equiv (S_{\delta}^{\tau})_i^{\mu}-(S_{\delta}^{\tau})_i^{6-31G},
    \end{align}
    donde $(S_{\delta}^{\tau})_i^{\mu}$ es la entropía informacional asociada a la base $\mu$ para el átomo $i$-ésimo, $i\in\{1,2,...,36\}$, o la molécula con $i$ enlaces, $i\in\{0,1,2,3,4\}$, $\tau\in\{N_e,1\}$ indica si la densidad electrónica se encuentra normalizada al número de electrones del sistema ($N_e$) o a la unidad, mientras que $\delta\in\{\rho,\Pi,T\}$ indica la densidad electrónica en el espacio de posiciones ($\rho$), momentos ($\Pi$) o la suma de ambas ($T$).
    
    Se tomó como referencia 6-31G al ser la base que considera menos propiedades electrónicas a comparación de las bases restantes (polarización, densidad electrónica difusa, correlación electrónica e interacciones de largo alcance), por lo que valores negativos de $\Delta (S_{\delta}^{\tau})^{6-31G}$ indicarían una calidad menor a aquella de la base $6-31G$. Al comparar diferentes bases, aquella que muestre el valor $\Delta (S_{\delta}^{\tau})^{6-31G}$ máximo para un átomo específico, sería la base con la mejor calidad para tal átomo. Por último, la cantidad $\Delta (S_{\delta}^{\tau})^{6-31G}$ para la base $6-31G$ valdrá cero para cualquier átomo o molécula por ser la referencia.

        El comportamiento de las $\Delta (S_{T}^{N_e})^{6-31G}$ para todas las bases que no son referencia muestran aparentes fluctuaciones alrededor del cero que van incrementando proporcionalmente con el número atómico, lo cual indica que las bases habrán de tener calidades semejantes para números atómicos cercanos a 1, mientras que discrepan notoriamente para aquellos cercanos a 36. Esto debido a que las bases describen con mayor dificultad a los átomos conforme el número atómico incrementa. Este evento sucede de manera semejante para todos los métodos restantes.
        
        Los resultados mostrados en las figuras \ref{fig: deltasum-n-tot} muestran que el uso de la suma de la entropía de Shannon $S_{T}^{N_e}$ podría no fungir como medida confiable para la calidad de las bases si se compara entre diferentes átomos, por ejemplo, el átomo con número atómico 29 en la figura \ref{fig: deltasum-mp3-n-tot} mostraría que la calidad en la base aug-cc-pVTZ es menor a aquella en la base 6-31G, pues $\Delta (S_{T}^{N_e})^{6-31G} = -1.23\pm0.015$, mientras que el átomo con número atómico 30 indicaría el caso contrario con $\Delta (S_{T}^{N_e})^{6-31G} = 2.147\pm0.011$.

            \sum_{i=1}^{36}\Delta (S_{T}^{N_e})_i^{6-31G} = \sum_{i=1}^{36}\Sqbr{(S_{T}^{N_e})_i^{\mu}-(S_{T}^{N_e})_i^{6-31G}},

No obstante, el orden que se presenta entre ellas y entre todas aquellas consideradas en el presente trabajo es algo que debe analizarse a mayor detalle, lo cual queda para trabajos futuros.
